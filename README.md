# ML Network Intrusion Detection System (NIDS)

## 📌 Problem Statement
Traditional intrusion detection systems struggle to identify new and evolving cyber threats. This project aims to develop a Machine Learning-based Network Intrusion Detection System that classifies network traffic as normal or malicious in real time.

---

## 🎯 Objectives

- Build a Machine Learning model to detect network intrusions.
- Analyze and preprocess network traffic datasets.
- Compare multiple ML algorithms and select the best performer.
- Develop a REST API for predictions.
- Create a dashboard to visualize network activity and alerts.
- Integrate real-time packet capture for live intrusion detection.

---

## ✨ Planned Features

- Data preprocessing & feature engineering
- Multiple ML model training and evaluation
- Real-time packet capture (Scapy/PyShark)
- Intrusion prediction API (FastAPI)
- Interactive React dashboard
- Attack history and logging
- Performance metrics & visualizations
- Model comparison and benchmarking
